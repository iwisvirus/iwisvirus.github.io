**SUCCESS!!**
- key to success is pw recovery speed
- depends on encryption type, hardware resources (local CPU, local GPU, network distributed compute)

**What hardware?**
1. CPU(Central Processing Unit)
   - Complex tasks, relatively low number of cores, relatively low parallel processing
2. GPU(Graphics Processing Unit)
   - simple tasks, relatively high number of cores, relatively high parellel processing
3. Distributed resources
- high-level requirements
  - more GPUS, the better
  - prioritization of CPU cores over clock speed
  - GPUs with more shading units are preferable
  - OpenCL 2.0 and higher
  - adequate cooling and power supply

**distributed pw recovery (Passware kit agents support)**
- password kit forensics distributes workload over a network of multiple windows or linux computers as well as amazon ec2 and microsoft azure
- passware kit agent should be installed on each machine and the same version as passware kit forensic
- each computer running passware kit agent can simultaneously support multiple CPUs and GPUs
- passware kit forensic includes 5 agents with the ability to purchase more
- supports all brute-force file types
