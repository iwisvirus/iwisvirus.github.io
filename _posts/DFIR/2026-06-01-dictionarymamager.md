**Dictionary**
- a list of words or combinations of chars
    - all words within a language
    - words compiled manually
    - previous passwords
    - all words from a forensic or/and memory image
    - downloaded from passware account
    - created using third party tools
- used for possible passwords, possible parts of passwords
- decrease number of passwords to check
- managed through dictionary manager
    - creates one dictionary from the selected dictionaries and remove duplicates

**How**
- create txt file
- compile from a memory image
- download from passware account or othe resources
- procedurally created using third-party tools
- added using a known passwords/part attack

**Password Exchange**
- opt in service
- provides access to list of pw found by Passware Kit Users Worldwide
- can be used within a dictionary attack
- can be updated manually office or automatically online
- pw uploaded anonymously
- can be used without internet
- pw, file type, attack type, attack duration are shared
