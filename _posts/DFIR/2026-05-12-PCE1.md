**Forensic Procedures**
- maintain copies of case data, 'working', 'archive'
- control and track exported material and decrypted versions of files to avoid cross-contamination
- work on cases not files
- collate all relevant material
- acquite memory image, all devices and clouds, as much data as possible
- work with dictionaries
  - index all evidence and export to dictionary file
  - organize and update dictionaries
  - use suitable dictionary files''
 
**password kit**
- do encryption triage and analysis
  - start attacking files with less secure encryption
  - re-use found passwords
  - verify patterns in passwords
- export and save decrypted files and logs
- clean up working space
